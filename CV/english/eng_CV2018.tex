\section{Personal Projects}

\cventry{October 2018}{AlphaZero Pytorch Implementation}{}{}{}{Open source PyTorch implementation of a single threaded self play reinforcement learning agent with Monte Carlo Tree Search (MCTS). The model was used to learn $n \times n$ tic-tac-toe with arbitrary size $n$.}

\cventry{August 2018}{Rubiks Cube Agent}{}{}{}{A Rubik's Cube reinforcement learning agent. Trained without any human knowledge to solve any arbitrary cube by learning to predict the outcome of a Monte Carlo Tree Search predicting the Q-value of a certain action. Learned to solve any $2 \times 2$ cube in an average time of 2.5 seconds.}

\cventry{July 2018}{Gland Instance Segmentation Through Overlapping Contour Regions and Random Transformation Sampling}{}{}{}{This paper was written during my employment at ContextVision AB. I developed an algorithm for instance segmentation of glands in hematoxylin and eosin stained medical images with limited data using random transformation sampling.}

\cventry{May 2018}{Improving Temporal Convolutional Networks}{}{}{}{I improved temporal convolutional neural networks and compared them to the traditional long short-term memory LSTM networks on the task of automatic image captioning.}

\cventry{Mars 2018}{Real-time Style Transfer}{}{}{}{A collaborative Python and JavaScript project where the content of a camera stream gets transferred into a given artistic style in real time using style transfer, convolutional neural networks and a website GUI.}
